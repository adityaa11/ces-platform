## Architecture enforcement

The architecture suite discovers every package under `packages/`, `adapters/`, and `apps/`; compares workspace imports with declared dependencies; rejects core-to-adapter/compiler dependencies; and detects dependency cycles. A synthetic core package with a Laravel-adapter dependency produces the expected violation, proving the guard fails intentionally bad architecture.

Production core source and the generated Policy Manifest are scanned for production-framework terminology. Repository CI runs the complete `pnpm check` on pinned Node and pnpm versions.
